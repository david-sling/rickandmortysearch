# Rick and Morty Search
Created using React JS

Search information about rick and morty characters

---

### To install and run:
* Clone repository
* `npm i `
* `npm start`

### To see live Demo:
* Click [here](https://rickandmortydemo.davidsling.in)
* <https://rickandmortydemo.davidsling.in>
